# mospoly_project

Условия:

1) Для использования сайта необходимо устройство ввода информации.
2) Человек должен быть, как минимум, зрящим, для использования полного функционала - сылшащим.

Инструкция:
&nbsp;Сверху любой страницы можно увидеть навигационную панель, благодаря которой можно перемещаться по сайту.
&nbsp;&nbsp;Логотип является ссылкой на главную страницу сайта:
&nbsp;&nbsp;&nbsp;По открытию сайта вы попадаете на эту страницу.
&nbsp;&nbsp;&nbsp;На главной странице вы можете увидеть карусель последних композиций, выложенных авторами.
&nbsp;&nbsp;&nbsp;Чтобы перейти на одну из композиций - нужно кликнуть по её названию.
&nbsp;&nbsp;Поисковая полоса не является рабочей.
&nbsp;&nbsp;По нажатии кнопки "Upload" вы попадаете на страницу загрузки композиции:
&nbsp;&nbsp;&nbsp;Функционал данной страницы доступен только зарегистрированным пользователям с флагом "Исполнитель".
&nbsp;&nbsp;&nbsp;Чтобы загрузить композицию нужно ввести ее название, выбрать файл который вы хотите загрузить и нажать кнопку "Done".
&nbsp;&nbsp;Кнопка "Зарегистрироваться" ведёт к странице с регистрацией:
&nbsp;&nbsp;&nbsp;На странице доступно три поля:
&nbsp;&nbsp;&nbsp;&nbsp;Поле ввода логина,
&nbsp;&nbsp;&nbsp;&nbsp;Поле ввода пароля,
&nbsp;&nbsp;&nbsp;&nbsp;Поле с двумя флагами, позволяющими выбрать тип своего аккаунта.
&nbsp;&nbsp;&nbsp;Так же на странице доступна кнопка "Done", по нажатию которой вы отправляете свои данные на сервер, зарегистрировавшись.
&nbsp;&nbsp;&nbsp;На странице изменяется кнопка "Зарегистрироваться" на "Логин", по нажатию которой вы можете перейти на страницу с авторизацией:
&nbsp;&nbsp;&nbsp;На странице доступно два поля: логин и пароль, по вводу которых вы сможете авторизоваться, нажав на кнопку "Done".
&nbsp;&nbsp;Страница композиции:
&nbsp;&nbsp;&nbsp;Доступна кнопка проигрывания музыки, по нажатию на которую воспроизводится музыка.
&nbsp;&nbsp;&nbsp;Так же доступна статистика прослушиваний и лайков композиции.
&nbsp;&nbsp;&nbsp;Доступна кнопка "Download", по нажатию на которую скачивается композиция, открытая раннее.
